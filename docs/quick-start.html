Introduction ​

If you've ever wanted an easier way to manage your MiSTer FPGA game library, search for games, apply filters, and launch them quickly from any device, you're in the right place.

MiSTer FPGA Smart Launcher consists of two applications:

  • Smart Scanner — a Windows application that scans your MiSTer FPGA game library and associates metadata with your games, enabling powerful search and filtering.
  • Smart Launcher — a web application installed on your MiSTer FPGA that lets you browse generated database, search, filter, and launch your games directly on MiSTer FPGA from any device with a web browser.

TIP

Although this guide may look long, the setup process is much simpler than it appears.

Follow each section in order and you'll be ready in just a few minutes.

TIP

For convenience, Smart Scanner and Smart Launcher will always be released together using the same version number

ROMs in ZIP archives ​

ROMs stored inside .zip archives are also supported.

INFO

Most MiSTer cores do not support launching ZIP files directly.

However, when using the standard MiSTer UI file browser, ZIP archives are displayed like folders. After selecting the ROM inside the archive, MiSTer automatically extracts it and passes it to the appropriate core.

Smart launcher do the same thing: When you will select a game to Play using Smart Launcher web app, the system will unpack the zip archive in a temporary folder and then will launch the rom using the required core

Smart Scanner ​

Smart Scanner is a Windows application that scans the ROM collection installed on your MiSTer FPGA.

During the scan, it can associate metadata with your games, making them searchable by title, genre, year, publisher, and more.

Once the scan is complete, Smart Scanner generates a database folder that can be copied directly to your MiSTer FPGA.

Ok, here we go -> Launch Smart Scanner and click Next to begin.

Add Your Systems ​

The first step let you decide which systems (or platforms) you want to scan.

Click Add System to add a new platform.

For each system you must specify:

  • The folder containing the ROMs.
  • The corresponding ROM path on your MiSTer FPGA.

There are two ways to select your ROM folder.

Option 1 — Local ROM Collection ​

If your PC contains the same ROM collection as your MiSTer FPGA, simply select the local folder.

Option 2 — Browse Your MiSTer Over the Network ​

If your MiSTer FPGA is connected to your local network, Smart Scanner can access it directly.

WARNING

This requires the Samba service to be enabled on your MiSTer FPGA.

You can download the script from here

WARNING

Browsing directly your MiSTer FPGA folders can drastically increase the time required to scan the ROM set and database generation.

Run the samba_on script from the Scripts menu on your MiSTer.

If the script is not available, download it and copy it into your Scripts folder before running it.

Once Samba is enabled, your MiSTer will be accessible from Windows File Explorer.

When selecting the ROM folder, enter inside the searchbar path the IP address of your MiSTer:

For example:

text
\\192.168.3.10

You will then be able to browse your MiSTer folders directly from Windows.

TIP

After configuring all your systems, click Export Profile to save your configuration.

You can import it later without repeating the setup process.

When you're finished, click Next.

Apply Metadata ​

Smart Scanner can associate metadata with every ROM.

You have three options:

  • No Metadata — Skip metadata generation.

  • PigSaint — Download and use the official PigSaint CSV metadata database.

  • User — Use your own CSV file.

WARNING

Without metadata, Smart Launcher's filtering features will not be available.

If you choose User, you can start from the provided CSV template.

The TAGS column is used to assign metadata to each ROM.

INFO

Custom tags must follow the same naming convention used by the PigSaint dataset.

Refer to the PigSaint documentation documentation for the complete list of supported tags and the naming convention to use.

Generate Database ​

After the scan completes, Smart Scanner generates a folder named database.

If you enable Override Existing Database, select the existing index.db file from your current database folder.

Smart Scanner will update the existing database instead of creating a completely new one.

Copy Your Database ​

Copy the generated database folder into your MiSTer FPGA Smart Launcher installation directory:

text
/media/fat/Scripts/MiSTer-Smart-Launcher

DANGER

Replace the existing database folder with the newly generated one, otherwise Smart Launcher will continue using the previous database.

Smart Launcher ​

MiSTer FPGA Smart Launcher is a web application for MiSTer FPGA that lets you browse your game library, search and filter your ROM collection, and launch games from any device with a web browser, including PCs, smartphones, and tablets.

The application runs a lightweight web server directly on your MiSTer FPGA, so no additional software is required on the client device.

Before You Begin ​

Smart Launcher relies on a database that contains information about your ROM collection.

Before installing Smart Launcher, you must generate this database using MiSTer FPGA Smart Scanner.

INFO

If you haven't created the database yet, follow the Smart Scanner guide before continuing with this installation.

Installation ​

After generating the database, follow these steps:

  1. Copy the MiSTer-Smart-Launcher folder to:

    text
    /media/fat/Scripts

  2. Copy the generated database folder into:

    text
    /media/fat/Scripts/MiSTer-Smart-Launcher

  3. On your MiSTer FPGA, open the Scripts menu.

  4. Enter the MiSTer-Smart-Launcher folder and launch the run.sh script.

Once the application is running, open the following address from any web browser on your local network:

text
http://<YOUR-MISTER-IP>:8282

For example:

text
http://192.168.3.10:8282

WARNING

Smart Launcher is not started automatically.

If you reboot or power off your MiSTer FPGA, you will need to run run.sh again. Automatic startup will be available in a future release, allowing Smart Launcher to run automatically whenever your MiSTer FPGA boots.

Launch a Game ​

Just search your desired game, click on it and then click on Play!
